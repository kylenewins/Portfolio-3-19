Burger API App (Node, Express, MySQL)

Although this App seems simple on the front-end, the back-end is where all of the magic happens. Upon loading the site, the burger names are pulled from an existing SQL database and appeneded to the appropriate list. As Users add, and in turn, eat their new burgers, the data is pushed through the API to the database on the back-end. A simple but effective demonstration of GET, PUT, POST, and DELETE AJAX methods.

FriendFinder App (Node/Express)

This App is a micro-simulation of sites like eHarmony and Match.com. The app asks users a few generic questions to determine their overall personality, and then compares those results to an existing array of "Friends" to compute their best match. As soon as users receive their results, they are added to the friend array so that future users can potentially match with them.

Reddit Jr. App (MongoDB/Mongoose)

This app is a simple web scraper that allows the user to search any subreddit and immediately be returned the titles and links associated with the most recent posts. It is, in essence, a smaller/lighter way of browsing reddit with added note saving capability. In addition, every time a user searches a subreddit, all of the results are stored in a mongoDB for later use. The notes are associated to the individual article in the db, so user's can return and view their notes at a later date.
